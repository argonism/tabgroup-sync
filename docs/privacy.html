Privacy Policy

Last updated: 2026-04-01

Overview

Tab Group Sync is a Chrome extension that synchronizes tab groups across multiple browser windows within the same profile. Your privacy is important to us. This policy explains what data the extension accesses and how it is handled.

Data Collection

This extension does not collect, transmit, or share any personal data.

  • No data is sent to external servers.
  • No analytics or tracking is included.
  • No user accounts or sign-ins are required.

Data Access

The extension accesses the following browser data solely to provide its functionality:

  • Tab Groups (tabGroups permission): Read and modify tab group properties (title, color, collapsed state) to synchronize them across windows.
  • Tabs (tabs permission): Read tab URLs and manage tabs within groups to mirror tab content across windows.
  • Storage (storage permission): Store user preferences (sync mode, whitelist/blacklist, sync property toggles) using chrome.storage.sync. This data is synced across the user's Chrome instances via their Google account, but is never sent to any third-party server.

Data Storage

User settings are stored in chrome.storage.sync, which is managed by Chrome and tied to the user's Google account. No other persistent storage is used. The extension does not use cookies, local databases, or external storage services.

Third-Party Services

This extension does not use any third-party services, APIs, or SDKs.
