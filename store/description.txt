Keep your tab groups in sync across every Chrome window — automatically.

If you use multiple Chrome windows, you know the pain: you organize your tabs into neat groups in one window, but your other windows are a mess. Tab Group Sync fixes that. Create a tab group in any window, and it instantly appears in all your other windows — same name, same color, same tabs, same order.

✦ What it syncs
• Group name and color
• Collapsed / expanded state
• Tabs inside the group (URLs)
• Tab order within the group
• Group creation and removal

Everything stays in sync in real time. Rename a group, change its color, add a tab, navigate to a new page — the change is mirrored everywhere.

✦ You're in control
• Auto mode — sync everything, no setup needed
• Whitelist — pick exactly which groups to sync
• Blacklist — sync everything except the ones you exclude
• Fine-grained toggles — turn off syncing for individual properties (e.g. sync name and color, but not collapsed state)
• Master on/off switch — pause syncing with one click
• Manual sync button — force a full sync whenever you want

✦ Privacy first
• Zero data leaves your browser. No servers, no analytics, no tracking.
• Settings are stored in Chrome's built-in sync storage — nothing else.
• Fully open source.

Works with any number of windows within the same Chrome profile. Requires no sign-in and no configuration to get started — just install and go.
